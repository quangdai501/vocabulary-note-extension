# Icons

This folder should contain the extension icons in the following sizes:

- **icon16.png** - 16x16 pixels (toolbar icon)
- **icon48.png** - 48x48 pixels (extension management page)
- **icon128.png** - 128x128 pixels (Chrome Web Store)

You can use any icon design tool or online generator to create these icons.

## Temporary Solution

For testing purposes, you can use placeholder images or create simple colored squares with the size requirements above.

## Recommended Tools

- [Favicon Generator](https://favicon.io/)
- [Icon Generator](https://www.icoconverter.com/)
- Figma, Adobe Illustrator, or any graphics editor
